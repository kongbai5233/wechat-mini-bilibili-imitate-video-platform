# 微信小程序 —— 短视频互动平台

## 📌 项目简介

本项目基于微信小程序框架开发，实现了一个仿 B 站风格的短视频浏览与社交互动平台。

系统结合微信云开发能力，完成视频播放、点赞收藏、文章发布、用户登录及数据同步等功能，构建了一个轻量化的视频内容与互动平台。

---

## 🚀 核心功能

### 🎬 视频模块

- 视频列表展示（缩略图、标题、作者信息）
- 视频播放页面
- 支持全屏播放
- 支持进度条拖动
- 点赞与收藏功能
- 弹幕发送功能
- 视频数据存储于云数据库
- 视频资源存储于云存储

---

### 📝 内容 / 空间模块

- 文章列表展示
- 文章详情查看
- 发布文章
- 图片上传至云存储
- 文章收藏与取消收藏
- 内容写入云数据库

---

### 👤 用户模块

- 微信一键登录（调用 wx 官方 API）
- 获取用户头像与昵称
- 收藏页面（基于本地缓存）
- 点赞页面（基于本地缓存）
- 根据内容 type 标签跳转至不同页面（视频 / 文章）

---

## 🏗 系统架构

**前端**
- 微信小程序原生框架
- Vant Weapp UI 组件库

**后端**
- 微信云开发
  - 云数据库
  - 云存储

**数据管理**
- 基于微信账号的数据同步
- 本地缓存存储点赞与收藏状态
- 模块化页面结构设计

---

## 🧠 解决的问题

### 1️⃣ 视频浏览与播放便捷性
无需下载独立 App，直接在微信内完成视频浏览与播放。

### 2️⃣ 社交互动能力不足
集成微信生态，实现点赞、收藏与内容分享功能。

### 3️⃣ 登录流程复杂
采用微信授权登录，减少注册流程，提高用户体验。

### 4️⃣ 多设备数据不同步
基于微信账号实现数据同步，保证跨设备使用体验一致。
